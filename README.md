# Приложение Chess Puzzle

Проект использует Next.js.

## Установка

Установите зависимости:

```bash
npm install
```

## Разработка

Запуск локального сервера разработки:

```bash
npm run dev
```

## Деплой

Vercel использует следующую команду:

```bash
npm run build
```

Команда `build` собирает приложение для деплоя.
